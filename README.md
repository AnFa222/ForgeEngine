# ForgeEngine

**ForgeEngine** is a lightweight **2D game engine written in Python** designed to help you build games and interactive experiences quickly and efficiently.

It includes:
- Game loop & event handling  
- Object & component system
- Input management 
- Collision detection  
- Window & rendering pipeline

---

##  Requirements

Make sure you have:

- Python 3.x  
- Pygame
- Pyinstaller (optional)

Install dependencies:

```bash
pip install pygame
pip install pyinstaller
```
